# Find Skills

This skill helps users discover and install agent skills.

## How to Search
- Use `npx skills find [query]` to search for skills
- Browse available skills at: https://skills.sh/

## Verification Before Recommending
- Check install count
- Verify source reputation
- Check GitHub stars

## Installing Skills
```bash
npx skills add <package> -g -y
```
